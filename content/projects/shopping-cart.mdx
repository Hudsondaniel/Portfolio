## Project Overview

CartBliss is a modern e-commerce shopping cart that focuses on user experience and clean design. Built with React and powered by Zustand for state management, it provides a smooth shopping experience with intuitive cart management and seamless checkout flow.

## Key Features

### 🛍️ Shopping Experience
- Product browsing with grid layout
- Featured products carousel
- Detailed product popups
- Dynamic price updates

### 🛒 Cart Management
- Add/remove products
- Quantity adjustments
- Cart total calculation
- Persistent cart state

### 💫 User Interface
- Responsive design for all devices
- Clean, modern aesthetics
- Smooth transitions
- Intuitive navigation

## Technical Highlights

```javascript
// Cart Store with Zustand
import create from 'zustand';

const useCartStore = create((set) => ({
  items: [],
  
  addItem: (product) => set((state) => ({
    items: [...state.items, { ...product, quantity: 1 }]
  })),

  removeItem: (productId) => set((state) => ({
    items: state.items.filter(item => item.id !== productId)
  })),

  updateQuantity: (productId, quantity) => set((state) => ({
    items: state.items.map(item => 
      item.id === productId 
        ? { ...item, quantity } 
        : item
    )
  })),

  clearCart: () => set({ items: [] }),

  getTotal: () => get().items.reduce((total, item) => 
    total + (item.price * item.quantity), 0
  )
}));

export default useCartStore;
```
